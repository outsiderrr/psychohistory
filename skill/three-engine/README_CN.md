# 三引擎分析框架（Three-Engine Analysis）

[English](./README.md) · **中文**

> 从三个互补的视角分析任何决策情境：博弈论 [GT]、心理模型 [PSY]、组织行为学 [ORG]。独立的原子 skill——可以单独使用，也可以被组合进更大的分析工作流。

## 做什么

输入一个决策情境（一个事件、一个模式、一个假设），从三个不同的角度产出结构化分析。三个引擎看到的东西不同——它们的交汇是强信号，它们的分歧揭示真正的不确定性。

## 使用场景

- **地缘政治**："俄罗斯为什么做 X？" / "伊朗下一步会怎样？"
- **商业战略**："CEO 会怎么回应这个竞争威胁？"
- **谈判分析**："交易空间在哪？"
- **新闻解读**："四国同周访华，背后什么逻辑？"
- **政策分析**："这个监管为什么被设计成这样？"
